# Python-Painting-Doraemon-Pikachu

使用 Python3 的 turtle 标准库绘制哆啦A梦、皮卡丘。
Use Python 3 Turtle graphics to draw Doraemon and Pikachu.

## Fork & Enhanced Edition by Ravinder Ozha

This repository is respectfully forked from the original project by PerpetualSmile.

## My Additions / 我的改进

* Added modified Doraemon version
* Bilingual comments (Chinese + English)
* Improved readability
* Animation experiments (eyes / blinking / fun effects)
* Beginner-friendly structure

## Files

* `Doraemon.py` → Original version
* `Doraemon_modified.py` → My enhanced version
* `Pikachu.py` → Original Pikachu drawing

## Run

```bash
python Doraemon.py
python Doraemon_modified.py
python Pikachu.py
```
